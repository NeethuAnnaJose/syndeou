# Syndeou Website

Website for Syndeou - Alpha Learning System (ALS)

## Setup

Install dependencies:
```bash
npm install
```

## Development

Run the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

## Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Tech Stack

- React 18
- React Router DOM
- Vite
- CSS (Blue and White theme)
